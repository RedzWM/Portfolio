## ðŸŽ¨ Customization

### Change Colors
Edit CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
}
```

### Add/Edit Content
Directly edit content in the `index.html` file

### Change Avatar Image
1. Save your photo as `images/avatar.jpg`
2. Recommended: 500x500px, JPG/PNG format
3. The image will automatically display as circular with animations

### Modify Animations
Adjust timing and effects in `script.js` and `styles.css`
